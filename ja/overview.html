オーディナルズ理論の概要

オーディナルズは、Satoshiの番号付けスキームであり、個別のsatを追跡したり転送したりすることを可能とします。これら番号は、オーディナル番号(ordinal numbers)と呼ばれます。 それぞれのSatoshiは、採掘された順に番号付けされ、トランザクションインプットからトランザクションアウトプットへ先入先出法に従って転送されます。この番号付けスキーム、および転送スキームは 順序(order) に依存、すなわち番号付けについては採掘された 順序 、 転送についてはトランザクションのインプットとアウトプットの 順序 に依存します。 これが、 オーディナルズ(=順序(序数)) と命名されるゆえんとなります。

技術的な詳細はthe BIPで確認することができます.

オーディナルズ理論は、分離されたトークンや、他のブロックチェーン、ビットコインに変更は必要ありません。まさに今機能しています。

オーディナル番号は、いくつかの異なる表現形式が存在します

  • 整数表記:2099994106992659オーディナル番号は、Satoshiが採掘された順番に割り当てられます。

  • 十進法表記: 3891094.16797最初の番号は、Satoshiが採掘されたブロック高、２番目はそのブロック内のSatoshiのオフセット

  • 度数表記: 3°111094′214″16797‴.後述します

  • パーセント表記: 99.99971949060254% .ビットコインの供給量からみたSatoshiの位置を、割合で表現

  • 名前: satoshi. オーディナル番号をアルファベットaからzで符号化したもの

NFT, セキュリティトークン、アカウント、ステーブルコインのような任意のアセットが、オーディナル番号を使って安定的なIDとして割り当てることができます。

オーディナルズはon GitHubにて開発されているオープンソースプロジェクトです。プロジェクトは、オーディナルズスキームのBIP, すべてのSatoshiの位置を追跡するためビットコインコアノードと通信するインデックス,オーディナルを考慮したトランザクション作成を可能とするウォレット、ブロックチェーンをインタラクティブに探索するためのエクスプローラ、Satoshiにデジタルアーティファクトを刻むことのできる機能、そしてこのマニュアルを含んでいます。

レアリティ

人間は誰しもコレクターです。そして、Satoshiがいま追跡・転送可能になったので、人々は自ずとこれらを集めたくなるでしょう。オーディナルズ論者は、どのsatがレアで欲しがられる対象なのか、決めることができます。そのためのヒントにはこのようなものがあります

ビットコインには周期的なイベントがあります。中には頻繁に起こるものもあれば、滅多に起きないものもあります。このイベントは自然とレアリティシステムへと結びつけることができます。周期的なイベントとは：

  • ブロック: 新しいブロックは、10分ごとに生成されます。

  • 難易度調整: 2016ブロックごとに、あるいはおよそ2週間ごとに、ビットコインネットワークは難易度ターゲットを調整することによってハッシュレートを変更します。

  • 半減期: 21万ブロックごとに、あるいはおよそ4年ごとに、ブロック内で新たに生み出されるSatの総量が半分になります。

  • サイクル: 6回の半減期ごとに、不思議なことが起こります。半減期と難易度調整が重なります。これはコンジャンクションと呼ばれます。コンジャンクションは、大体24年ごとに発生します。最初のコンジャンクションは2032年に起こる見込みです。

これにより、下記のレアリティレベルを定義することができます：

  • コモン: ブロック内の最初のsatを除くすべてのsat
  • アンコモン: 各ブロックの最初のsat
  • レア: 各難易度調整の最初のsat
  • エピック: 各半減周期の最初のsat
  • レジェンダリ: 各サイクルの最初のsat
  • ミシック: ジェネシスブロックの最初のsat

これらは、度数表記で表すことができます。度数表記は、satoshiのレアリティを一目で確認することのできる曖昧性のない表現です。

│ │ │ ╰─ Index of sat in the block
│ │ ╰─── Index of block in difficulty adjustment period
│ ╰───── Index of block in halving epoch
╰─────── Cycle, numbered starting from 0

名前

それぞれのsatoshiには名前があり、文字AからZを使って、採掘されるsatoshiが未来に行けば行くほど短くなるように考えられています。最初が短い名前で、未来に行くほど長い名前になるように考えることも可能で、それは問題ないのですが、意味を持ちやすい短い名前が、消費されることのないジェネシスブロックに固定さてしまいます。

例えば、1905530482684727°'の名前は"iaiufjszmoba".最後に発掘されるsatoshiは "a"です。10文字以下の全ての組み合わせはいつか世に出回ります。

エキゾチック

Satoshi は名前や、レアリティといった理由以外からも評価されるかもしれません。これは、二乗根,三乗根のような整数であるといった、番号それ自身の性質によるものかもしれません。または、SegWitが有効化されたブロック477120内のものや最後に採掘される2099999997689999° といった、歴史的なイベントに紐づく数字であるかもしれません。

このようなsatoshiは "エキゾチック" と名づけられます。どのsatoshiがエキゾキックなのか、なにがそうさせるのかは、主観的です。オーディナルズ論者は、彼らが発明した基準に基づいてエキゾチックを追い求めることを推奨しています。

碑文

Satoshiには任意のコンテンツを刻印することができ、ビットコインネイティブのデジタルアーティファクトを作成することができます。刻印は刻印されるsatoshiをトランザクションで送信し、オンチェーンで刻銘内容を明らかにすることで行われます。このコンテンツは、satoshiと表裏一体となり、追跡、譲渡、保管、売買、紛失、再発見が可能な普遍のデジタルアーティファクトとなります。

考古学

初期のNFTをカタログ化し収集することに専念している考古学者たちの活気あるコミュニティが誕生しました。Chainleftによる歴史的なNFTの素晴らしい概要

初期のNFTの一般的に受け入れられている境界線は、2018年3月19日です。この日は、最初のERC-721コントラクトであるSU SQUARES,がイーサリアムにデプロイされた日です。

オーディナルズがNFT考古学者にとって興味深いかどうかは未解決の問題です！ある意味では、オーディナルズは2022年初頭にオーディナルズ仕様が完成したときに作成されました。この意味では、それは歴史的な興味を引くものではありません。

しかし別の意味では、オーディナルズは2009年にサトシ・ナカモトがビットコインのジェネシスブロックを採掘したときに実際に作成されました。この意味では、オーディナルズ、特に初期のオーディナルズは確かに歴史的な興味を引くものです。

多くのオーディナルズ論者は後者の見解を支持しています。その理由の一つは、オーディナルズが少なくとも2回、現代のNFT時代が始まるずっと前に独立して発見されたからです。

2012年8月21日、Charlie Leeはビットコインにプルーフ・オブ・ステークを追加する提案をBitcoin Talkフォーラムに投稿しました。そして実装されましたがデプロイされることはありませんでした。

2012年10月8日、jl2012は同じフォーラムに小数表記を使用し、オーディナルズの重要な特性をすべて備えたスキームを投稿しました。このスキームは議論されましたが、実装されませんでした。

これらのオーディナルズの独立した発明は、ある意味でオーディナルズが発見された、または再発見されたものであり、発明されたものではないことを示しています。オーディナルズはビットコインの数学の必然性であり、それは現代の文書化からではなく、その古代の起源から生じています。それらは、多くの年前に最初のブロックの採掘と共に開始された一連の出来事の頂点なのです。
